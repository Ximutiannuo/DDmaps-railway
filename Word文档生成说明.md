## 如何重新生成Word文档

如果您修改了Markdown源文件（`司机端使用说明书_俄语版.md`），需要重新生成Word文档：

### 方法1：使用转换脚本（推荐）

1. 确保已安装 `python-docx` 库：
   ```bash
   pip install python-docx
   ```

2. 运行转换脚本：
   ```bash
   python convert_md_to_docx.py
   ```

3. 生成的Word文档将保存为 `司机端使用说明书_俄语版.docx`

### 方法2：手动转换

1. 打开 `司机端使用说明书_俄语版.md` 文件
2. 使用Markdown编辑器（如Typora、Mark Text）或在线工具转换为Word
3. 或直接复制内容到Word中，然后手动调整格式
